PROGRAMACION FUNCIONAL EN PYTHON.

La programación funcional es un paradigma de programación que se centra en tratar a las funciones como elementos de primer nivel y evitar los cambios de estado y las mutaciones de datos. En Python, la programación funcional se puede implementar utilizando varias características del lenguaje:

Funciones de orden superior: Python permite pasar funciones como argumentos a otras funciones, así como devolver funciones como resultado. Esto permite operar con funciones de manera flexible y poder aplicar conceptos como map(), filter() y reduce().

Funciones lambda: Son funciones anónimas que se pueden usar para operaciones simples y pasarse como argumentos a otras funciones.

Comprensión de listas y generadores: Estas características permiten crear listas y generadores de manera concisa aplicando transformaciones o filtrado a elementos de una secuencia.

Inmutabilidad: Aunque Python no es puramente funcional, se pueden emplear técnicas para minimizar la mutabilidad de los datos, como evitar cambios directos en estructuras de datos y preferir la creación de nuevas instancias.
